# UI System

## Aesthetic
Shared Base industrial-neon:
- Dark Web3 default.
- Space Mono display typography with IBM Plex Sans body text.
- Blue primary action, acid-lime highlights, green success, amber review state.
- Atmospheric grid and scanline treatment instead of flat backgrounds.

## Component Rules
- Cards and panels use 8px radius maximum.
- Buttons use lucide icons and minimum 44px touch targets.
- Page layout uses CSS Grid; component internals use Flexbox.
- Focus states use visible acid-lime outlines.
- Hover states lift important cards and actions.

## Product Fit
VaultAI uses this system as an app console, not a marketing landing page. The first viewport shows operational state, actions, agent flow, and MCP tools.
